# FBcmd4J
Certificado de java


## Instalación
**Clonar el repositorio**  
```
git clone https://github.com/kevinguajardo/FBcmd4J.git
```

**Importar a NetBeans**  
1. Dar clic en `Team -> Git -> Clone`.
2. Seleccionar la carpeta raíz donde se clonó el proyecto.
